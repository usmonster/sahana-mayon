DokuWiki_Plugin::getInfo (  ) 

General Info.

Needs to return a associative array with the following values:

author - Author of the plugin email - Email address to contact the author date - Last modified date of the plugin in YYYY-MM-DD format name - Name of the plugin desc - Short description of the plugin (Text only) url - Website with more information on the plugin (eg. syntax description)
