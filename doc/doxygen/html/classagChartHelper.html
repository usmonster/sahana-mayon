$  chartId,
array $  dataArray 
) [static]

A method to set chart data.

It automatically checks whether or not the data file exists and, if-so, whether the current data file matches the passed data array (avoiding unnecessary writes). If no data file exists it attempts to create one.

Parameters:
mixed $uniqueIdent A string (unique) identifier for a chart
string $chartId One of the CHART_ constants
array $dataArray An array of chart data
Returns:
boolean Returns false if the chart is not known to the class
