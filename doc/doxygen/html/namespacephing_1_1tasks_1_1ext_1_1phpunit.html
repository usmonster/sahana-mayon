Detailed Description

Scans a list of files given by the fileset attribute, extracts all subclasses of PHPUnit(2)_Framework_TestCase / PHPUnit(2)_Framework_TestSuite.

Various utility functions.

Simple Testrunner for PHPUnit2/3 that runs all tests of a testsuite.

Transform a PHPUnit2 xml report using XSLT.

Prints XML output of the test to a specified Writer.

Prints short summary output of the test to Phing's logging system.

A wrapper for the implementations of PHPUnit2ResultFormatter.
