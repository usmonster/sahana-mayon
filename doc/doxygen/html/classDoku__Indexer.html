Doku_Indexer::lookupKey ( $  key,
&$  value,
$  func = null 
)

Find pages containing a metadata key.

The metadata values are compared as case-sensitive strings. Pass a callback function that returns true or false to use a different comparison function. The function will be called with the $value being searched for as the first argument, and the word in the index as the second argument. The function preg_match can be used directly if the values are regexes.

Parameters:
string $key name of the metadata key to look for
string $value search term to look for, must be a string or array of strings
callback $func comparison function
Returns:
array lists with page names, keys are query values if $value is array

Tom N Harris <tnharris@whoopdedo.org>
