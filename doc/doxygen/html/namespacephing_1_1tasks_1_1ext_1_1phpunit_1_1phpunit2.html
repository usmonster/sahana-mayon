phing::tasks::ext::phpunit::phpunit2 Namespace Reference

This abstract class describes classes that format the results of a PHPUnit2 testrun. More...

Detailed Description

This abstract class describes classes that format the results of a PHPUnit2 testrun.

Prints XML output of the test to a specified Writer.

Prints short summary output of the test to Phing's logging system.

Prints plain text output of the test to a specified Writer.
