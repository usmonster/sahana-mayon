00152 to speed things up. Matrix algebra is a great testing ground for ideas about time and memory 
00153 performance optimation.  Keep in perspective that PHP JAMA scripts are still plenty fast for use as 
00154 a tool for learning about matrix algebra and quickly extending your knowledge with new scripts
