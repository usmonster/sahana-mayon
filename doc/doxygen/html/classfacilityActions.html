facilityActions::executeDelete ( sfWebRequest $  request  ) 

executeDelete()

Delete a facility record. Redirects to facility list page when it is done.

Parameters:
$request sfWebRequest object for current page request

Locate the associated agEntity record through the associated agSite record. Call the agEntity object's delete() method, which results in the deletion of all of the records associated with this agFacility record, including contact information.

agEntity will call its agSite object's delete() method which will, in turn, call our agFacility object's delete() method. The agFacility object will also delete the associated agFacilityResource records.

If there was no related agEntity record found, something is wrong.

Redirect to facility/list when done.
