sfWidgetFormSchema::setNameFormat ( $  format  ) 

Sets the format string for the name HTML attribute.

If you are using the form framework with symfony, do not use a reserved word in the name format. If you do, symfony may act in an unexpected manner.

For symfony 1.1+, the following words are reserved and must NOT be used as the name format:

* module (example: module[s]) * action (example: action[s])

However, you CAN use other variations, such as actions[s] (note the s).

Parameters:
string $format The format string (must contain a s for the name placeholder)
Returns:
sfWidget The current widget instance
Exceptions:
InvalidArgumentException when no s exists in the format
