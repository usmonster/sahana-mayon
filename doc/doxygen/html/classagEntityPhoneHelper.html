Method to return entity phones for a group of entity ids, sorted from highest priority to lowest priority.

Parameters:
array$entityIdsA single dimension array of entityId's that will be queried.
boolean$strTypeBoolean that determines whether or not the phone contact type will be returned as an ID value or its string equivalent.
boolean$primaryBoolean that determines whether or not only the primary phone will be returned (for that type).
string$phoneHelperMethodThe phone helper method that will be called to format or process the phones. This should be an agPhoneHelper::PHN_GET_* constant. If left NULL, only phone ID's will be returned.
array$phoneArgsAn array of arguments to pass forward to the phone helper.
Returns:
array A three dimensional array, by entityId, then indexed from highest priority phone to lowest, with a third dimension containing the email type as index[0], and the phone value as index[1].
