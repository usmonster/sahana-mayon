DepthSelector::isSelected ( PhingFile $  basedir,
$  filename,
PhingFile $  file

The heart of the matter.

This is where the selector gets to decide on the inclusion of a file in a particular fileset. Most of the work for this selector is offloaded into SelectorUtils, a static class that provides the same services for both FilenameSelector and DirectoryScanner.

Parameters:
basedir the base directory the scan is being done from
filename is the name of the file to check
file is a PhingFile object the selector can use
Returns:
whether the file should be selected or not
