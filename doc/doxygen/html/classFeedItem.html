FeedItem::$additionalElements = Array()

Any additional elements to include as an assiciated array.

All $key => $value pairs will be included unencoded in the feed item in the form < $key>="">$value</> Again: No encoding will be used! This means you can invalidate or enhance the feed if $value contains markup. This may be abused to embed tags not implemented by the FeedCreator class used.
