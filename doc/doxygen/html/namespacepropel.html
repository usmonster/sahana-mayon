Detailed Description

Propel's main resource pool and initialization & configuration class.

PropelConfigurationIterator is used internally by PropelConfiguration to build a flat array from nesting configuration arrays.

PropelConfiguration is a container for all Propel's configuration data.

The base class of all exceptions thrown by Propel.

This static class is used to handle Propel initialization and to maintain all of the open database connections and instantiated database maps.
