dokuwiki_xmlrpc_server::dokuwiki_xmlrpc_server (  ) 

Constructor.

Register methods and run Server

Trigger XMLRPC_CALLBACK_REGISTER, action plugins can use this event to extend the XMLRPC interface and register their own callbacks.

Event data: The XMLRPC server object:

$event->data->addCallback() - register a callback, the second paramter has to be of the form "plugin:<pluginname>:<plugin method>"

$event->data->callbacks - an array which holds all awaylable callbacks
