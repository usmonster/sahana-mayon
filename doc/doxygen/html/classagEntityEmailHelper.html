$  strType = NULL,
$  primary = NULL 
)

Method to return entity emails for a group of entity ids, sorted from highest priority to lowest priority.

Parameters:
array$entityIdsA single dimension array of entityId's that will be queried.
boolean$strTypeBoolean that determines whether or not the email contact type will be returned as an ID value or its string equivalent.
boolean$primaryBoolean that determines whether or not only the primary email will be returned (for that type).
Returns:
array A three dimensional array, by entityId, then indexed from highest priority email to lowest, with a third dimension containing the email type as index[0], and the email value as index[1].
