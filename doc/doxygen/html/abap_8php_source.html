00014  *
00015  * Reference abap language documentation (abap 7.1) : http://help.sap.com/abapdocu/en/ABENABAP_INDEX.htm
00016  *
00017  * ABAP syntax is highly complex, several problems could not be addressed, see TODO below if you dare ;-)
00018  * Be aware that in ABAP language, keywords may be composed of several tokens,
00019  *    separated by one or more spaces or carriage returns
00020  *    (for example CONCATENATE 'hello' 'world' INTO string SEPARATED  BY ' ')
00021  *    it's why we must decode them with REGEXPS. As there are many keywords with several tokens,
00022  *    I had to create a separate section in the code to simplify the reading.
00023  * Be aware that some words may be highlighted several times like for "ref to data", which is first
00024  *    highlighted for "ref to data", then secondly for "ref to". It is very important to
00025  *    position "ref to" after "ref to data" otherwise "data" wouldn't be highlighted because
00026  *    of the previous highlight.
00027  * Control, declarative and other statements are assigned URLs to sap documentation website:
00028  *    http://help.sap.com/abapdocu/en/ABAP<statement_name>.htm
00029  *
00030  * CHANGES
00031  * -------

00036  * 2007/06/27 (1.0.0)
00037  *   -  First Release
