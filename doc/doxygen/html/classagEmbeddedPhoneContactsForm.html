This sets up the agEmbeddedPhoneContactForms nedded to create an agPerson.

Depending on whether the person is being created or edited, forms are either empty (if creating) or populated (if editing, and the person has, through the entity table an agEntityPhoneContact value for the agPhoneContactType associated with the embedded form.
