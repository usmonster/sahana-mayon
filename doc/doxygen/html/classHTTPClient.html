Detailed Description

This class implements a basic HTTP client.

It supports POST and GET, Proxy usage, basic authentication, handles cookies and referers. It is based upon the httpclient function from the VideoDB project.

HTTPClient::sendRequest ( $  url,
$  data = '',
$  method = 'GET' 
)

Send an HTTP request.

This method handles the whole HTTP communication. It respects set proxy settings, builds the request headers, follows redirects and parses the response.

Post data should be passed as associative array. When passed as string it will be sent as is. You will need to setup your own Content-Type header then.

Parameters:
string $url - the complete URL
mixed $data - the post data either as array or raw data
string $method - HTTP Method usually GET or POST.
Returns:
bool - true on success
