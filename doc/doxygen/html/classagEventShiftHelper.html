static agEventShiftHelper::returnSingleCurrentFacilityResourceShifts ( $  eventId,
$  staffed = NULL,
$  timestamp = NULL,
$  minEnd = TRUE 
) [static]

Method to return an associative array keyed by event facility resource id with a value of one of the current shifts of the facility resource.

Because there can be many "current" shifts, only one is returned. Parameters $staffed and $minEnd limit whether or not only staffed shifts will be returned or whether or not the returned shift will be among those "current shifts" that ended first or last.

Parameters:
integer(4) $eventId The event being queried.
boolean $staffed An optional parameter to determine whether or not to restrict results to those that are staffed (TRUE) or not staffed (FALSE). By default this parameter is ignored.
timestamp $time A optional value that adjusted the concept of 'current' from the application's CURRENT_TIMESTAMP (or PHP time()) to the passed value (essentially enabling the user to ask what shifts will be active at this point-in-$time
boolean $minEnd Parameter to determine whether the "first shift" returned will also be the first or last to end. If $minEnd = TRUE (the default), the the returned shift will be among those that ended first.
Returns:
array A key value array of ($event_facility_resource_id => $shift_id)
