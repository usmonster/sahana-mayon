sfFormPropel::processValues ( $  values  ) 

Processes cleaned up values with user defined methods.

To process a value before it is used by the updateObject() method, you need to define an updateXXXColumn() method where XXX is the PHP name of the column.

The method must return the processed value or false to remove the value from the array of cleaned up values.
