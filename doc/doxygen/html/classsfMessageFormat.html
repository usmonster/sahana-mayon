Formats the string.

That is, for a particular string find the corresponding translation. Variable subsitution is performed for the $args parameter. A different catalogue can be specified using the $catalogue parameter. The output charset is determined by $this->getCharset();

Parameters:
string $string the string to translate.
array $args a list of string to substitute.
string $catalogue get the translation from a particular message
string $charset charset, the input AND output charset catalogue.
