====== Organization Management Walk-Through ======

===== Editing an Organization =====
It's helpful to keep organization information updated so any references to that organization are current.  As a user you'll navigate from the list, through viewing the organization information, and then to the edit screen.

===== Walk-Through =====
After click the ID number in the list you'll be viewing the Organization's information.

{{:manual:user:organizations:orgview.png?350|Click Edit}}

Click the Edit button to move to the edit screen.  On the edit screen you can update the Name and Description information, much like when creating an Organization.  After you update the information, click "Save and Continue" to be sure it is stored.  Click "Next" below to continue.
